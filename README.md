# OpenClaw Starter Pack

Starter pack para bootstrap de um fluxo inspirado no OpenClaw em outro workspace ou equipe.

## O que tem aqui

- `MANUAL-DO-ZERO.md`: guia principal para partir do zero
- `IMPLEMENTATION-CHECKLIST.md`: checklist de implantacao
- `ROLLOUT-BY-MATURITY.md`: roteiro por nivel de maturidade
- `PACK-CONTENTS.md`: inventario do pacote
- `RUNTIME-GENERATION.md`: como gerar runtime a partir de templates

## Diretorios principais

- `templates/`: templates de workspace, repo, scheduler e `.openclaw`
- `scripts/`: scaffolding e renderizacao de templates
- `examples/`: exemplo ponta a ponta de workspace inicial

## Ordem recomendada

1. Leia `MANUAL-DO-ZERO.md`.
2. Passe por `IMPLEMENTATION-CHECKLIST.md`.
3. Adapte os arquivos em `templates/`.
4. Use `examples/` como referencia de estrutura final.

## Observacao

Este repositorio publica o conteudo extraido do starter pack local, em vez do arquivo `.zip`.
