# AcmeOrdersApi

Repositorio piloto ficticio para demonstrar como conectar um repo ao fluxo OpenClaw.

## Objetivo

- receber tarefas pequenas de manutencao
- validar localmente
- produzir artefato humano simples

## Estrutura minima

- `src/`: codigo da aplicacao
- `tests/`: validacao minima
- `docs/`: notas locais do repo
- `AGENTS.md`: regra local do repositorio
